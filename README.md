# zephyr

yarn dev:zephyr ./examples/zephyr/bubble.zph 
yarn dev:zephyr ./examples/zephyr/sieve.zph 

--emit-bc

Требования на 3:
1. константы
2. базовые типы
3. агрегаты(массивы[n][m], структуры)
4. Классы(структуры и методы)
5. Функции(параметры по ссылке, кол-во фиксированно)
6. Библиотеки(нативные - read, print, printf, приведение типов as)
7. Ключевые слова - зарезервированные идентификаторы
8. Операторы(присваивания, сравнения)
9. Условия
10. Циклы
11. Операторные скобки
12. read, write

Требования на 4:
1. Классы: наследование и полиморфизм
2. Библиотека
3. Методы работы с элементами классов
